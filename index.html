Introduction à l'informatique DÉBUTANT

Comprendre ce qu'est un ordinateur, son fonctionnement interne et le vocabulaire de base de l'informatique. Ce module pose les bases essentielles pour toute la suite du parcours.
1
Définition de l'informatique : science du traitement automatique de l'information. Comprendre la notion de données (data) et de programme.
2
Les composants physiques (hardware) : processeur (CPU), mémoire vive (RAM), disque dur (HDD/SSD), carte mère, carte graphique (GPU), alimentation.
3
Les logiciels (software) : système d'exploitation (Windows, Linux, macOS), applications, drivers. Différence hardware vs software.
4
Le cycle de traitement : Entrée → Traitement → Sortie → Stockage. Exemples concrets avec clavier, CPU, écran.
5
Les unités de mesure : bit, octet (byte), Ko, Mo, Go, To. Calculs de conversion et exemples pratiques (photos, vidéos, etc.).
6
Les types d'ordinateurs : PC de bureau, laptop, serveur, tablette, microcontrôleur. Cas d'utilisation de chacun.
🎯
Mini-Projet Dresser une fiche descriptive complète d'un ordinateur réel : identifier et nommer tous les composants visibles, noter les caractéristiques (RAM, stockage, processeur) et calculer la capacité totale en Go.
02

Systèmes d'exploitation DÉBUTANT

Maîtriser l'interface et les fonctions fondamentales d'un système d'exploitation. Apprendre à gérer fichiers, dossiers, paramètres et raccourcis pour travailler efficacement au quotidien.
1
Rôle d'un OS : gestion du matériel, des processus, des fichiers et des utilisateurs. Présentation de Windows 10/11, Ubuntu Linux, macOS.
2
Bureau, barre des tâches, gestionnaire de fichiers. Navigation dans l'arborescence : C:\Users\ (Windows) ou /home/ (Linux).
3
Créer, renommer, déplacer, copier, supprimer des fichiers et dossiers. Comprendre les extensions de fichiers : .txt, .pdf, .jpg, .exe, etc.
4
Panneau de configuration / Paramètres : résolution d'écran, comptes utilisateurs, mises à jour, gestion des applications installées.
5
Raccourcis clavier indispensables : Ctrl+C/V/X/Z, Alt+Tab, Win+E, Ctrl+Alt+Del. Augmente la productivité de 40%.
6
Gestionnaire des tâches : surveiller CPU, RAM, disque. Terminer un processus bloqué. Identifier les programmes qui ralentissent le PC.
7
Introduction au terminal / invite de commandes : commandes cd, ls/dir, mkdir, del/rm. Puissance de la ligne de commande.
🎯
Mini-Projet Organiser un espace de travail complet : créer une arborescence de dossiers (Travail > Projets > [Année] > [Mois]), y placer des fichiers types, puis réaliser les mêmes opérations en ligne de commande.
03

Internet & Réseaux DÉBUTANT

Comprendre comment fonctionne Internet, les protocoles de communication, les adresses IP, et naviguer de manière efficace et sécurisée sur le Web.
1
Qu'est-ce qu'un réseau ? LAN (réseau local), WAN (réseau étendu), Internet. Notion de client/serveur. Comment les données voyagent en paquets.
2
Adresse IP : IPv4 (192.168.1.1) et IPv6. Adresse MAC. Rôle du routeur, du modem, du switch. Connexion Wi-Fi vs Ethernet.
3
Protocoles essentiels : HTTP/HTTPS, FTP, DNS, DHCP. Comment un navigateur charge une page web (DNS → IP → TCP → HTML).
4
Les navigateurs web : Chrome, Firefox, Edge. Onglets, favoris, historique, mode privé. Moteurs de recherche et opérateurs avancés ("", site:, filetype:).
5
Sécurité en ligne : reconnaître HTTPS (cadenas), phishing, mots de passe forts, double authentification (2FA). Gestionnaires de mots de passe.
6
Email professionnel : rédiger, CC/BCC, pièces jointes, signatures. Différence Gmail, Outlook, Thunderbird. Organiser sa boîte mail avec des filtres et dossiers.
7
Cloud computing : Google Drive, OneDrive, Dropbox. Synchronisation, partage de fichiers, accès multi-appareils. Avantages du cloud vs stockage local.
🎯
Mini-Projet Configurer un environnement numérique personnel : créer un compte email professionnel, organiser sa boîte mail, stocker 5 documents importants sur le cloud et les partager avec un lien, puis tester sa connexion réseau avec ping et ipconfig.

HTML & CSS — Bases du Web INTERMÉDIAIRE

Créer vos premières pages web avec HTML pour la structure et CSS pour le style. Comprendre comment les sites internet sont construits de zéro, sans framework.
1
Structure d'un document HTML : <!DOCTYPE html>, <html>, <head>, <body>. Balises sémantiques : <header>, <main>, <footer>, <section>, <article>.
2
Texte et liens : <h1>–<h6>, <p>, <strong>, <em>, <a href="">. Images : <img src="" alt="">. Listes : <ul>, <ol>, <li>.
3
Formulaires HTML : <form>, <input>, <textarea>, <select>, <button>. Attributs type, name, placeholder, required.
4
CSS — sélecteurs : tag, .classe, #id, combinateurs. Propriétés fondamentales : color, background, font-size, margin, padding, border.
5
Le modèle de boîte (box model) : content → padding → border → margin. box-sizing: border-box. Positionnement : static, relative, absolute, fixed.
6
Mise en page moderne : Flexbox (display:flex, justify-content, align-items) et CSS Grid (display:grid, grid-template-columns). Responsive design avec @media.
7
Variables CSS (:root), animations CSS (@keyframes, transition), pseudo-classes (:hover, :focus) et pseudo-éléments (::before, ::after).
🎯
Mini-Projet Créer une page de portfolio personnel complète : une section hero avec nom et présentation, une grille de compétences en CSS Grid, un formulaire de contact fonctionnel, et un design responsive qui s'adapte au mobile et au desktop.

JavaScript — Programmation Web INTERMÉDIAIRE

Rendre vos pages web interactives et dynamiques avec JavaScript. Du DOM à la manipulation d'événements, en passant par les APIs, ce module est la clé pour créer des sites vivants.
1
Bases du JS : variables (var, let, const), types (string, number, boolean, null, undefined), opérateurs. console.log() pour déboguer.
2
Structures de contrôle : if/else, switch, boucles for, while, for...of. Fonctions : déclaration, paramètres, return, fonctions fléchées (=>).
3
Tableaux et objets : .push(), .map(), .filter(), .reduce(). Manipulation d'objets : accès propriétés, déstructuration, spread operator ....
4
Le DOM (Document Object Model) : document.getElementById(), querySelector(). Modifier contenu (innerHTML, textContent), style (style.color), classes (classList).
5
Événements : addEventListener('click', fn), 'input', 'submit', 'keydown'. Objet event, preventDefault(). Délégation d'événements.
6
JavaScript asynchrone : setTimeout, setInterval. Promises (.then().catch()). async/await. Fetch API pour charger des données JSON depuis une URL.
7
LocalStorage : localStorage.setItem(), getItem(), removeItem(). Sauvegarder et récupérer des données côté navigateur. JSON.stringify / JSON.parse.
🎯
Mini-Projet Créer une application "Liste de tâches" (To-Do List) complète : ajouter/supprimer/cocher des tâches, filtrer par statut, sauvegarder dans LocalStorage pour que les données persistent après rechargement. Interface stylée avec CSS.

Bases de données & SQL INTERMÉDIAIRE

Comprendre comment les données sont organisées, stockées et interrogées. SQL est le langage universel des bases de données relationnelles, utilisé dans presque toutes les applications.
1
Qu'est-ce qu'une base de données ? Base de données relationnelle vs NoSQL. Notions de table, ligne (enregistrement), colonne (champ), clé primaire, clé étrangère.
2
Installation de SQLite ou MySQL. Créer une base de données et une table : CREATE DATABASE, CREATE TABLE avec types (INT, VARCHAR, TEXT, DATE, BOOLEAN).
3
Opérations CRUD : INSERT INTO, SELECT * FROM, UPDATE ... SET, DELETE FROM. Clause WHERE, ORDER BY, LIMIT.
4
Requêtes avancées : JOIN (INNER, LEFT, RIGHT) pour relier plusieurs tables. GROUP BY, HAVING. Fonctions d'agrégation : COUNT(), SUM(), AVG(), MAX(), MIN().
5
Index et performances : pourquoi créer un INDEX. EXPLAIN pour analyser une requête. Normalisation : 1NF, 2NF, 3NF pour éviter la redondance.
6
Introduction aux bases NoSQL : MongoDB (documents JSON), comparaison avec SQL. Quand choisir l'un ou l'autre selon le projet.
🎯
Mini-Projet Concevoir et créer une base de données pour une bibliothèque scolaire : tables Livres, Auteurs, Élèves, Emprunts. Relier les tables avec des clés étrangères, insérer 10 enregistrements par table, puis écrire des requêtes pour afficher les livres empruntés, les élèves en retard, et les auteurs les plus populaires.

Cybersécurité & Protection des données AVANCÉ

Comprendre les menaces informatiques modernes, protéger ses systèmes et ses données personnelles. Ce module couvre les bonnes pratiques indispensables pour tout utilisateur ou professionnel du numérique.
1
Les principales menaces : virus, malware, ransomware, spyware, adware, cheval de Troie. Comment ils se propagent (email, téléchargement, USB, réseau).
2
Attaques courantes : phishing, spear phishing, man-in-the-middle, injection SQL, XSS (Cross-Site Scripting), brute force. Exemples réels et comment les reconnaître.
3
Mots de passe : règles d'un mot de passe fort (12+ caractères, majuscule, chiffre, symbole). Gestionnaires de mots de passe (Bitwarden, 1Password). Double authentification (TOTP, SMS, clé physique).
4
Chiffrement : symétrique (AES) vs asymétrique (RSA). HTTPS et certificats SSL/TLS. Chiffrer ses fichiers avec VeraCrypt ou BitLocker. VPN : rôle et limites.
5
Sécurité réseau : firewall (pare-feu), DMZ, segmentation réseau. Scan de ports avec nmap. Analyser le trafic réseau avec Wireshark (introduction).
6
Règlementation et éthique : RGPD (Europe), protection des données personnelles, droits des utilisateurs. Différence hacker éthique (white hat) vs malveillant (black hat). Bug bounty.
7
Plan de réponse aux incidents : que faire si on est hacké ? Isoler, analyser, restaurer, signaler. Importance des sauvegardes 3-2-1 (3 copies, 2 supports, 1 hors site).
🎯
Mini-Projet Réaliser un audit de sécurité personnel : vérifier si ses emails ont été compromis (haveibeenpwned.com), changer tous les mots de passe faibles, activer la 2FA sur ses comptes principaux, chiffrer un dossier sensible avec VeraCrypt, et rédiger une charte de sécurité numérique personnelle.
09

Maintenance & Dépannage informatique AVANCÉ

Savoir diagnostiquer et résoudre les pannes matérielles et logicielles. Maintenir un ordinateur en bonne santé, optimiser ses performances et gérer les mises à jour système.
1
Diagnostic de panne : méthode de déduction (quoi ? quand ? comment ?). Lecture des messages d'erreur (BSOD Windows, kernel panic Linux). Journaux système (Event Viewer, journalctl).
2
Maintenance préventive : nettoyage physique (poussière, ventilateurs), renouvellement de la pâte thermique, vérification des températures avec HWMonitor ou sensors.
3
Optimisation logicielle : désactiver les démarrages automatiques, nettoyer le registre Windows (CCleaner), défragmenter un HDD, TRIM sur SSD. Libérer l'espace disque.
4
Gestion des drivers : mettre à jour les pilotes (GPU, réseau, son). Résoudre les conflits de drivers. Utiliser le mode sans échec pour déboguer.
5
Réinstallation du système : créer une clé USB bootable (Rufus), partitionner le disque, installer Windows/Ubuntu proprement, restaurer ses données depuis une sauvegarde.
6
Résolution de problèmes réseau : ping, tracert/traceroute, ipconfig/ifconfig, nslookup. Réinitialiser la pile TCP/IP. Configurer une IP fixe.
7
Virtualisation : créer et gérer des machines virtuelles avec VirtualBox ou VMware. Utiliser des snapshots pour tester sans risque. Comprendre les conteneurs Docker (introduction).
🎯
Mini-Projet Créer une machine virtuelle Ubuntu avec VirtualBox, installer un serveur web Apache dessus (sudo apt install apache2), configurer une page HTML personnalisée accessible depuis l'hôte via l'adresse IP de la VM. Documenter toutes les étapes dans un rapport de maintenance.

Projet Final — Application Web Complète EXPERT

Mettre en pratique TOUS les concepts du parcours en créant une application web complète de A à Z : front-end, back-end, base de données, sécurité, déploiement. C'est votre chef-d'œuvre.
1
Conception : définir les fonctionnalités, créer les wireframes (Figma ou papier), modéliser la base de données (schéma entité-relation), choisir la stack technique.
2
Base de données : créer les tables SQL (utilisateurs, contenus, etc.), relations, index. Rédiger les requêtes CRUD. Préparer les données de test.
3
Back-end : créer une API REST simple avec Python (Flask) ou Node.js (Express). Endpoints : GET /articles, POST /articles, PUT /articles/:id, DELETE /articles/:id.
4
Front-end : interface HTML/CSS/JS responsive. Consommer l'API avec fetch(). Afficher dynamiquement les données. Formulaires avec validation côté client.
5
Sécurité : hacher les mots de passe (bcrypt), authentification par sessions ou JWT, protection contre les injections SQL (requêtes préparées) et XSS (échappement des entrées).
6
Tests & Débogage : tester les endpoints avec Postman, tester l'interface dans plusieurs navigateurs, corriger les bugs. Rédiger des tests unitaires basiques.
7
Déploiement : héberger sur un VPS (DigitalOcean, Render, ou Railway), configurer un nom de domaine, activer HTTPS avec Let's Encrypt. Votre application est en ligne et accessible au monde entier 🚀
🏆
Projet Final Créer et déployer une plateforme de blog avec inscription/connexion des utilisateurs, publication d'articles avec éditeur riche, système de commentaires, tableau de bord administrateur, et interface responsive. Présenter le projet avec une démo live et un rapport technique de 5 pages.
